The goal of the R package `MoveR` is to help users analyze the output of
automated video tracking solutions in a reproducible, reliable and open
framework.

## Installation

You can install the development version from
[GitHub](https://github.com/) with:

``` r
# install.packages("remotes")
remotes::install_github("qpetitjean/MoveR")
```

Then you can attach the package `MoveR`:

``` r
library("MoveR")
```
